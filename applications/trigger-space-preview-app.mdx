title: "Trigger Space Preview Application"
sidebarTitle: "Trigger Space Preview App"
description: "A [Trigger](/stack/trigger) workflow for creating or deleting a [Google Cloud Run](/stack/google-cloud-run) service for [Space](/applications/space) application when a GitHub pull request is (re)opened or closed."
---
It adds a comment to the pull request with the corresponding URL used to access the preview application.

It only (re)deploys the service when there are changes to **Nx** projects with the `space` [tag](https://nx.dev/core-features/enforce-project-boundaries#tags).
